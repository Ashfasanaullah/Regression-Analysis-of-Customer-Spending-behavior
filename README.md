# Regression Analysis of Customer Spending Behavior

This project demonstrates both simple and advanced regression techniques.

## Contents:
- Linear Regression
- Polynomial Regression (Degree 3 & 10)
- Multivariable Polynomial Regression (3D visualization)

## Dataset:
Mall Customers dataset

## Tools:
Python, Pandas, Scikit-learn, Matplotlib

## Objective:
To understand relationship between income and spending behavior using regression models.

## Key Insight:
Higher degree polynomial models capture non-linear patterns but may lead to overfitting.
